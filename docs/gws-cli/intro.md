# GWS CLI 소개

## GWS CLI가 뭔가요?

**GWS CLI**(Google Workspace CLI)는 구글 워크스페이스의 모든 기능을 **터미널 명령어 하나로** 다룰 수 있게 해주는 도구입니다.

> 쉽게 말하면, **구글 워크스페이스의 만능 리모컨**이에요! 🎮
>
> TV 리모컨으로 채널을 바꾸고 볼륨을 조절하는 것처럼, GWS CLI로 구글 드라이브 파일을 찾고, 이메일을 보내고, 캘린더 일정을 확인할 수 있어요. 다만 리모컨 버튼 대신 **키보드로 명령어를 입력**하는 거죠.

```bash
# 구글 드라이브에서 파일 5개 보기 — 이게 전부입니다!
gws drive files list --params '{"pageSize": 5}'
```

## 뭘 할 수 있나요?

GWS CLI 하나면 구글 워크스페이스의 거의 모든 서비스를 다룰 수 있습니다:

| 서비스 | 할 수 있는 것 | 비유 |
|--------|-------------|------|
| **Google Drive** | 파일 검색, 업로드, 다운로드, 공유 | 서랍 정리하기 |
| **Gmail** | 이메일 보내기, 읽기, 검색 | 우체통 관리하기 |
| **Google Calendar** | 일정 보기, 만들기, 수정하기 | 비서에게 스케줄 물어보기 |
| **Google Sheets** | 데이터 읽기, 쓰기, 수정하기 | 엑셀 편집하기 |
| **Google Docs** | 문서 만들기, 편집하기 | 워드 작업하기 |
| **Google Chat** | 메시지 보내기, 읽기 | 메신저 사용하기 |

그리고 **100개 이상의 AI 에이전트 스킬**도 포함되어 있어서, AI 도구와 함께 쓰면 더욱 강력해집니다.

## 왜 CLI를 쓰나요?

"구글 드라이브 웹사이트에서 하면 되는데, 왜 굳이 터미널에서 해요?"

좋은 질문이에요! CLI를 쓰면 이런 점이 좋습니다:

### 1. 자동화가 가능해요

:::tip 비유: 자동 세탁기 vs 손빨래
매일 아침 "어제 온 이메일 중 중요한 것 요약해줘"를 손으로 하나하나 할 수도 있지만, CLI 명령어 하나로 **자동으로** 처리할 수 있어요. 손빨래 대신 세탁기를 쓰는 것과 같아요!
:::

```bash
# 매일 아침 실행하면 자동으로 스탠드업 리포트 생성
gws workflow +standup-report
```

### 2. AI 에이전트와 연동할 수 있어요

Claude Code 같은 AI 도구가 GWS CLI를 **도구로 사용**할 수 있습니다. "내 이메일에서 오늘 회의 관련 내용 찾아줘"라고 AI에게 말하면, AI가 알아서 GWS CLI 명령을 실행해요.

### 3. 반복 작업을 줄여줘요

매주 보고서를 만들 때마다 구글 시트 열고, 데이터 복사하고, 이메일로 보내는 과정을 **명령어 한 줄**로 끝낼 수 있어요.

## 이 가이드를 따라하면 무엇이 되나요?

이 가이드를 끝까지 따라하면 여러분은:

:::info 기대 결과
1. **GWS CLI를 설치하고 인증 설정**을 완료할 수 있어요
2. **터미널에서 구글 드라이브, 지메일, 캘린더, 시트**를 자유롭게 다룰 수 있어요
3. **Claude Code와 GWS CLI를 연동**해서 AI 비서처럼 활용할 수 있어요
4. **실무에서 반복 작업을 자동화**하는 워크플로우를 만들 수 있어요
:::
